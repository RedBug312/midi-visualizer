# MIDI Visualizer
MIDI visualize_midi written in Gtk and Gstreamer

## Build Environment

### Linux

```bash
$ sudo apt install gstreamer1.0-plugins-bad python3-pip ffmpeg libffi-dev
$ pip3 install --user gizeh moviepy mido intervaltree
```
